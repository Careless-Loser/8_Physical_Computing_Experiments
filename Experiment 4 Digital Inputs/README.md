# Experiment 4: Digital Inputs Overview

This experiment explores reading digital signals from the physical world by using push switches to control an output device, such as an LED.

## Description

Digital inputs allow a microcontroller to detect simple state changes, like whether a button is currently being pressed or released. This experiment covers how to properly connect push-button switches to the digital pins on the Arduino and use them to read input states (HIGH or LOW). By interpreting these signals, you will use conditional logic to turn a single red LED on and off, demonstrating the fundamental mechanics of interactive control systems. 

## Components Required

* 1x Arduino Uno R3
* 1x Breadboard
* 7x M-M wires (Male to Male jumper wires)
* 1x 5mm Red LED
* 1x 220 ohm resistor
* 2x Push switches
